文章写作

创建新文章

  • 功能: 创建一个新的 Markdown 文件，默认放置在 source/_posts 目录。
  • 示例: hexo new "我的第一篇文章"

创建草稿

  • 功能: 创建一个新的 Markdown 文件，默认放置在 source/_drafts 目录。
  • 示例: hexo new draft "未完成的草稿"

网页生成

生成静态文件

启动本地服务器

  • 功能: 启动本地开发服务器，默认在 http://localhost:4000 上访问。
  • 示例: 运行后在浏览器中访问 http://localhost:4000 查看本地预览。

查看草稿
