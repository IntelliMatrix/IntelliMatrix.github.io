至此，已经确认电脑上安装了 OpenSSH 相关程序

在 VScode 上安装 SSH - remote 拓展

这个拓展用于管理通过 SSH 连接的服务器

在 Extensions 中搜索 SSH - remote ，进行安装。

按照以下操作步骤添加服务器

  • Press F1 and run the Remote-SSH: Open SSH Host… command.
  • Enter your user and host/IP in the following format in the input box that appears and press enter: user@host-or-ip or user@domain@host-or-ip
  • If prompted, enter your password (but we suggest setting up key based authentication).
  • After you are connected, use File > Open Folder to open a folder on the host.

设置登录账户

如果有时候出现明明输入了正确的密码但是无法登录的情况，有可能是因为登录的是默认的管理员账户，这是我们可以通过更改 SSH 配置文件的方式指定登录账户。

无密码登录

无密码主要使用 SSH 生成的密钥，将其分别放在服务器上和 VScode 上，在登录中使用密钥即可跳过输入密码的步骤。

  • MobeXterm
  • WinSCP

接下来所有的提示全部使用 Enter 跳过。

之后在文件资源管理器中开启隐藏文件可见，打开 C:\Users\username\.ssh 路径，可以看到两个文件为 id_rsa 和 id_rsa.pub, 这分别对应私钥和公钥。私钥储存在操作终端上，而公钥要放置在服务器上。

在我进行本操作的过程中，曾尝试过将密钥生成在其他文件夹，而并非默认的路径 C:\Users\username\.ssh，但是这会导致使用 VScode 进行 SSH 连接时无法连接上，因此若没有对于 VScode 的运用较为熟练的情况下建议使用默认的路径保存。
