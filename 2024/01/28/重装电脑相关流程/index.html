重装电脑相关流程

这篇文章主要供我自己记录在电脑重装时候怎么恢复到自己最顺手的环境，可能会不时更新

重置电脑

建议从微软官方下载对应版本的电脑系统

https://www.microsoft.com/zh-tw/software-download/windows11

  • 笔记本默认已经激活Windows系统的话如果安装其他版本的系统有可能会失效，比如自带 Windows 11 中国大陆简体中文版，但是尝试安装 Windows 11 非中国大陆版会导致系统激活失效
  • 若使用系统 Setting 中的 Reset 有可能会导致 Windows Security Centre 无法正常打开，这一定可能和笔记本开发商尝试用自己的安全系统取代 Windows 系统的安全系统导致的文件缺失有关。

文件体系结构

所有个人文件都储存在用户名下的 Document 文件夹，Document 文件夹中包含两个重要文件

  • Document
    • PersonalFile
    • Software

PersonalFile 以个人的生活学习情景为划分进一步细分为 University、Blog、Music、Painting 等文件夹，在这些文件夹下则以事件为划分。

Software 文件夹则用来存放 Portable 软件，如 Geek。

资源管理器

  • 打开隐藏文件可视
  • 打开文件扩展名可视
  • 文件夹选项中取消勾选 显示最近使用的文件、显示常用文件夹、显示来自 Office.com 的文件
  • 快速访问栏常驻 Desktop、Picture、PersonalFile、Software，System（ PersonalFile 下路径，用来存放壁纸，SSH配置等系统相关文件），然后是 PersonalFile 下的常用领域文件，剩余为当前事件。

系统简化

下载 Geek，这是一款用来卸载东西的软件。

  • 微软OfficePLUS
  • Xbox 相关软件
  • Tips
  • Solitaire Collection
  • Quick Assist
  • Microsoft News
  • Microsoft OneDrive
  • Clipchamp - Video Editor

相关软件安装

基本常用软件

  • Chrome

  • QQ

  • 微信

  • 百度网盘

  • 网易云音乐

  • VScode

  • Office三件套 + Visio //通过 Office Tool Plus 安装

  • PotPlayer

  • Keepass

  • 7-zip

  • Calibre

  • Everything

  • Typora

  • Sumatra PDF

  • Acrobat

可选软件

  • Acrobat
  • Photoshop
  • Clip Studio
  • FL studio
  • Xmind

Portable Apps

需要通过快捷方式绑定到桌面上

  • Clash
  • FSCapture
  • Geek

美化

任务栏设置

  • 关闭 小组件
  • 关闭 Copilot
  • 搜索 设置为 搜索图标和标签
  • 将常用系统托盘图标打开
  • 任务栏对齐方式 靠左

开始菜单设置

  • 在 显示在电源按钮旁边的“开始”菜单上显示的文件夹 中设置打开除 网络 和 个人文件夹 外的其他选项
  • 关闭 显示最近添加的应用
  • 关闭 显示最常用的应用
  • 关闭 显示有关提示、快捷方式、新应用等的建议

Apps 应用打开位置

任务栏

  • 任务视窗
  • Everything
  • Keepass
  • FSCapture
  • 文件资源管理器
  • 任务管理器
  • Chrome
  • QQ
  • 微信
  • 网易云音乐
  • VScode
